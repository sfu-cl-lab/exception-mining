This package contains classes which are utilities of general usefulness to the class in the other packages of Tetrad IV.

Note that this package (tetrad.util) should have no dependencies on any other packages within the Tetrad IV application. Violations which need to be fixed include:

  • GSquareTest2.java - There are problems with GSquareTest.java, and as soon as the bugs are fixed, it should be substituted for GSquareTest2.java.
